# Vue Forms App

A Vue.js application for managing health holiday forms with multiple steps.

## Tech Stack
- Vue.js 3
- Vite
- Vueform
- Typescript

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Project Structure

- `src/components/` - Form components (ContactForm, ParticipantForm, ExtrasForm, etc.)
- `src/App.vue` - Main application component
- `src/main.js` - Application entry point
